🔐 Encryption and Decryption Project
This project demonstrates basic encryption and decryption techniques to secure data. It converts plain text into encrypted text and then decrypts it back to its original form.
-- Features
🔑 Encrypt plain text
🔓 Decrypt cipher text
🧠 Simple and easy logic
⚡ Fast execution
🔍 How It Works
Encryption: Converts plain text into cipher text
Decryption: Converts cipher text back to original text
